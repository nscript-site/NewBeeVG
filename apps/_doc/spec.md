
# 口播脚本规则

口播脚本为 markdown 格式。规则如下：

标题可以不播

正文中的控制指令不播。控制指令独占一行。

存在两种控制指令，一类是设定当前语音和指令控制，格式为: [@Voice(Instructions)]，也可以不带指令控制，格式为 [@Voice]。下面例子为正确的控制指令:

```markdown
[@Nla]

[@Nla(快速)]
```

其中，`[@Nla]` 代表使用的 Voice 代号是 `Nla`，无控制指令。`[@Nla(快速)]` 代表使用的 Voice 代号为 `Nla`，控制指令为 `快速`。

如果存在新的语音和指令控制指令，则用新的替换旧的。

另一个控制指令是 clip 控制指令。假定当前 clip 的 name，格式为 `[#Name]`，比如，`[#1]` 代表当前 clip 的 name 为 `1`。

从 clip 控制指令到下一个 clip 控制指令（或到片尾）的正文为这段 clip 的口播内容。
